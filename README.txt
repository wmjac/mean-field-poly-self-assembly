README

Please directly execute the file exact_coex_V1.py to obtain predictions from our model.
The parameters are m, Np, Ns, dw, ep. Please directly change them in the file for a specific setting.

One or two files will be created:
The .npy file is the free energy landscape (fig.2b). For phase separation cases, a .txt file containing information of coexisting phases will be generated.

Please find the exemplary case in the directory.
